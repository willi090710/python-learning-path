## Variables

**A variable stores data.**

### Example

```python
my_name = "Willi"
print(my_name)
```

### Output

```text
Willi
```

**You can also change the data stored in a variable.**

### Example

```python
number = 10
number = 20
print(number)
```

### Output

```text
20
```

## Data Types

**There are 4 important data types.**

### String (`str`)

```python
name = "Willi"
```

### Integer (`int`)

```python
number = 15
```

### Float (`float`)

```python
decimal_number = 6.356
```

### Boolean (`bool`)

```python
is_it_a_lie = False
```

**To check the data type of a variable, use `type()`.**

### Example

```python
name = "Willi"
print(type(name))
```

### Output

```text
<class 'str'>
```
